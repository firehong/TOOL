> 主要集成微信H5支付，APP支付，以及企业转制，企业红包，退款等功能

### 1.1.1 目录结构介绍
>weixinpay---微信app支付、H5支付、企业微信转账到余额、微信退款

  --config 微信支付相关配置参数

    &nbsp;&nbsp;&nbsp;--SignUtil 微信支付签名工具类

  --WxConfig.java 微信支付、退款等初始化参数设置
  --WxPay.java 微信支付请求类
